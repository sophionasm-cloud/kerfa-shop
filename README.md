# ☕ KERFA - Coffee Export & Trade Platform

A full-stack web application for Ethiopian coffee export and trade management. Built with Vue.js 3 frontend and Laravel 13 backend.

## 🚀 Live Demo

https://app-45694296-daec-4264-9167-45a65f5e680e.cleverapps.io/

## 📋 Features

### Admin Features
- ✅ Admin Login/Logout
- ✅ Dashboard with statistics (Products, Orders, Pending, Completed)
- ✅ Add, Edit, Delete Products
- ✅ Product image upload
- ✅ View and Manage Orders
- ✅ Update Order Status (Pending → Approved → Completed)

### User Features
- ✅ User Registration & Login
- ✅ Browse Products
- ✅ Add to Cart
- ✅ Place Orders
- ✅ Track Orders
- ✅ View Order History

### Public Features
- ✅ Product catalog
- ✅ Services pages (Global Export, Local Market, Premium Quality)
- ✅ Multi-language support (English/Amharic)
